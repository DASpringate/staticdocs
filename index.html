staticdocs 0.1

Convert package rd files to static html pages, suitable for serving on a website.

  • staticdocs-package: Build complete static documentation for a package.
  • build_package: Build complete static documentation for a package.
  • build_topics: Generate all topic pages for a package.
  • build_vignettes: List all package vignettes.
  • package_info: Return information about a package
  • parse_rd: Parse an rd file in to staticdocs format.
  • render_template: Render template.
  • to_html: Convert an rdoc to a list of html components.
